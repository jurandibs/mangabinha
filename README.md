# 🏗️ Mangabinha Locações - Landing Page

Bem-vindo ao repositório do **Mangabinha Locações**, uma landing page moderna, performática e interativa desenvolvida para uma empresa de aluguel de maquinários e equipamentos para construção civil.

## 🌟 Visão Geral

O projeto foi construído focando em **experiência do usuário (UX)** e **conversão**, permitindo que os clientes naveguem pelo catálogo de equipamentos, escolham as quantidades desejadas e enviem um pedido de orçamento automatizado diretamente para o WhatsApp da empresa.

## ✨ Principais Funcionalidades

- **Catálogo Interativo:** Grid dinâmico de equipamentos com suporte a filtro por categorias e pesquisa de itens em tempo real.
- **Orçamento via WhatsApp:** Sistema inteligente que coleta os itens selecionados (com quantidades) e redireciona o cliente para o WhatsApp com uma mensagem pronta e formatada.
- **Design Responsivo & Premium:** Interface de alto contraste e layout fluido, adaptando-se perfeitamente a dispositivos móveis, tablets e desktops.
- **Mapa Integrado:** Integração com Google Maps, destacando o endereço físico exato da empresa (Central de Retirada).
- **Sem Dependências Complexas:** Construído com Vanilla JavaScript e Tailwind CSS, garantindo máxima velocidade de carregamento sem necessidade de configurações de build.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica e acessível.
- **Tailwind CSS:** Estilização utilitária rápida e consistente (via CDN).
- **Vanilla JavaScript:** Lógica de interface, manipulação de DOM e integração com WhatsApp.
- **Google Material Symbols:** Ícones vetoriais modernos.

## 🚀 Como Executar o Projeto

Como se trata de um projeto front-end puramente estático, rodar a aplicação é extremamente simples:

1. Clone este repositório:
   ```bash
   git clone https://github.com/jurandibs/mangabinha.git
   ```

2. Navegue até a pasta do projeto:
   ```bash
   cd magabinha
   ```

3. Abra o arquivo `index.html` diretamente no seu navegador **OU** inicie um servidor local para melhor visualização de assets:
   ```bash
   # Utilizando Python
   python -m http.server 8080
   ```

4. Acesse `http://localhost:8080` no seu navegador.
